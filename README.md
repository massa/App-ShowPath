NAME
====

App::ShowPath - show the contents of the environment variable $PATH, in a browseable table form

SYNOPSIS
========

```bash
$ showpath
```

DESCRIPTION
===========

App::ShowPath is a simple script to show the contents of $PATH via Term::TablePrint;
